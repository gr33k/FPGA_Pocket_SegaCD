# Credits

FPGA_Pocket_SegaCD is an Analogue Pocket/openFPGA project targeting Sega Genesis first, with planned Sega CD support.

This project is based on and/or derived from the openFPGA Genesis work by opengateware:

- https://github.com/opengateware/openFPGA-Genesis

That project is itself a port of a MiSTer/fpgagen-derived Genesis core for Analogue Pocket.

Original and upstream work includes contributions from:

- Gregory Estrade / Torlus, original fpgagen project
- MiSTer Genesis core contributors
